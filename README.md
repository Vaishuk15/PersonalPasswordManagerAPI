## Prerequisites

Ensure you have the following installed on your machine:

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- [Docker]
- [Visual Studio 2022](https://visualstudio.microsoft.com/) or another compatible IDE (optional)
